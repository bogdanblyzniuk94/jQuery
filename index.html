Let's talk about jQuery!

jQuery is a fast, small, and feature-rich JavaScript library.
It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.
With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

Downloading and connect jQuery

You can download jQuery library from official site jQuery and add it to directory your project

                
    <script src="js/jquery-3.4.1.js" type="text/javascript"></script>

Or you can copy link to jQuery library from official site jQuery or other's CDN, for example, Google CDN

                
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>

Function $()

One of the most commonly used functions in jQuery is the $() function.

                
    var text = $("#password").val();

Also, you can call function $(), write jQuery instead $

                
    var text = jQuery("#password").val();

Select elements on web page

You can select items by ID,...

                
    var text = jQuery("#counter").html();

... class...

                
    var text = jQuery(".row").html();

... or tag

                
    var text = jQuery("div").html();

Manipulations

You can manipulate classes on web page with jQuery. This code allows you to change the class of the element of the web page when you enter 1 or 2.

                
    var div = prompt("1 or 2?");
    while (div !=1 && div !=2) {
        alert("Please, enter 1 or 2")
        div = prompt("1 or 2?");
    }
    $("#div").attr("class", "div_" + div);
    alert($("#div").get(0).className);

Also you can manipulate styles on web page. This code allows you to change styles of the selected element.

                
    $("#div").css("font-size", "25px");
    $("#div").css("color", "red");

    $("#div").css("background-color", function() {
        return "blue";
    });

    $("#div").css("width", "250px");

    $("#div").css("height", function() {
        var height = prompt("Enter height: ");
        return height + "px";
    });

jQuery functions

There are many jQuery functions. Here is some of them:
.html() - Get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element.
.text() - Get the combined text contents of each element in the set of matched elements, including their descendants, or set the text contents of the matched elements.
.clone() - Create a deep copy of the set of matched elements.
.append() - Insert content, specified by the parameter, to the end of each element in the set of matched elements.
.remove() - Remove the set of matched elements from the DOM.
.insertAfter() - Insert every element in the set of matched elements after the target.
.insertBefore() - Insert every element in the set of matched elements before the target.
Full list of jQuery functions available on jQuery API Documentation.
